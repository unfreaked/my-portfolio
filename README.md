# Personal Portfolio Website

A modern, responsive portfolio website built with React and Material-UI.

## Features

- Responsive design
- Dark/Light mode
- Smooth animations
- Project showcase
- Skills section
- Contact form
- Experience timeline
- Education section

## Setup Instructions

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Build for production:
```bash
npm run build
```

## Technologies Used

- React
- Material-UI
- Framer Motion
- Styled Components
- React Router
